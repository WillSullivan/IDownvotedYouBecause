    I downvoted you because...

    This website is designed to help people understand why you downvoted them.

    Participation in a reputation-ranked online community comes with its ups and downs. You post something and get upvoted when people like or agree with it. Unfortunately, you may also be downvoted because someone didn't agree or didn't like it.

    The goal for IDownvotedYouBecause is to allow users an easy, productive and helpful way to let someone know what the problem was that caused your downvote, what they can do to fix it, and why you think they should do so.

    You can help make this website better!

    This website is hosted on GitHub and served via GitHub Pages. This means that, if you have a GitHub account, you can request changes, updates, and new pages that cover different situations. You can even add them yourself by forking the website's repository, making the changes as you see fit. Once completed, you can send me, Will Sullivan (@willSullivan), a pull request to add your changes to the website.
